# korean-text-to-anki-deck
Generating word lists from articles (korean, export to Anki)

# Installation

Install konpy
[http://konlpy.org/en/v0.4.4/install/](http://konlpy.org/en/v0.4.4/install/)

Run script with:
python create_word_list
